# 雷轟（Raigo） セットアップ（初期化）定義書

## 1. 事前準備：奥義駒の選択
手番を決定する前に、各プレイヤーは自身の **奥義駒 (`hand3`)** を選択する。
- **選択対象:** `material.inc.php` の `piece_types` のうち、以下の条件を全て満たす駒。
  - `available` が `true` である。
  - `weight` が `9` である。
- **ルール:** 上記リストから **最大3個** までの駒を選び、自身の `hand3` コンテナに格納する。

## 2. 手番の決定
- 先手（First Player）と後手（Second Player）をランダムに決定する。

## 3. 隠駒 (On-goma) のセット
- **ターンプレイヤー:** 中央線より下に配置。
- **非手番プレイヤー:** 中央線より上に配置。
- 初期状態では先手プレイヤーの側に配置される。

## 4. 駒の構成と生成
- `material.inc.php` の `count` プロパティに基づき、駒を生成する。
- **総数:** 実質 **64枚**。
- **配置:**
  - 63枚を山札（雷山 / `deck`）にセットし、シャッフルする。
  - **最後の1枚:** 山札には入れず、直接 **除外駒 (`exclusion`)** コンテナにセットする。

## 5. 初期配布とプレイヤーの選択
山札から駒を選択し、初期配置を行う。

- **先手 (First Player):**
  1. 山札から **1枚** 選択して自身の **雲 (`kumo`)** に置く。
  2. 雲にある駒を自身の **体内 (`inside`)** または **手 (`hand`)** のいずれかに配置する。
- **後手 (Second Player):**
  1. 山札から **2枚** 選択して自身の **雲 (`kumo`)** に置く。
  2. 雲にある駒を自身の **体内 (`inside`)** または **手 (`hand`)** のいずれかに（それぞれ個別に）配置する。

## 6. 初期得点
- 双方のプレイヤーの初期得点を **2点** とする。

## 7. その他コンテナの初期状態
- **桜門 (Oumon):** 空（0枚）。
- **レコード (Record):** 空。
- **峡谷 (Kyoukoku):** 空。
- **頭 (Head):** 空。
